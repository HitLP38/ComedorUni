## Notas de uso

- Las citas en texto siguen el estilo APA 7 en su forma *(Autor, año)* o *(Autor, año, p. X)* cuando se realiza cita directa.
- Las obras con tres o más autores se citan con el formato *(Primer autor et al., año)* desde la primera aparición.
- Las normas internacionales se citan con la organización emisora como autor institucional.
- Las referencias a documentos normativos de Internet (RFC) se registran mencionando al IETF como emisor.
- Este archivo crece con cada avance documental; no se eliminan referencias sin justificación.
